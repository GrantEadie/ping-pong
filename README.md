## Description

_This program will take a user inputted number and output from 1 to the user inputted number. The program will then replace all numbers divisible by 3 with "ping", replace all numbers divisible by 5 with "pong" and replace all numbers divisible by 5 and 3 with "ping-pong"._

_Spec 1: Program will create a string of numbers from 1 to the user inputted number and output these numbers_
_Spec 2: Program will replace all numbers divisible by 3 with the word "ping"_
_Spec 3: Program will replace all numbers divisible by 5 with the word "pong"_
_Spec 4: Program will replace all numbers divisible by 3 and 5 with the word "ping-pong"_
